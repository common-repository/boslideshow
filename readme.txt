Plugin for cycling image gallery.

== Description ==

Add images to directory:

1. Upload images;
2. Set the size, order and timout of the slider;
3. Edit a text to be shown on each image;

Boslideshow can be viewed at <a href="http://www.bohemiawebsites.com/WORDPRESS">this page</a>.

You can <a href="http://www.bohemiawebsites.com/component/option,com_jdownloads/Itemid,163/catid,13/task,viewcategory/">download from our website.</a>

== Installation ==

It's easy to install.

You can <a href="http://www.bohemiawebsites.com/component/option,com_jdownloads/Itemid,163/catid,13/task,viewcategory/">download from our website.</a>

1. Extract `boslideshow.zip` and upload the folder `boslideshow` to the `/wp-content/plugins/` directory;
2. Activate the plugin through the `Plugins` menu in WordPress
3. Place something like `<?php if (function_exists(boslideshow_show())) { boslideshow_show(); } ?>` in your templates within the desired div tag.
4. Set images to display in the admin panel. Images are located in 'files' folder.
